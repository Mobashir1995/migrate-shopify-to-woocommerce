== Description ==
Shopify to WooCommerce Migration plugin will migrate all of your shopify product and categories to your WooCommerce Store. You have need to create a private app on your shopify store, allow required access and copy the store url, API Key, API Password and paste those credenstials into your <strong>"WordPress Admin > Shopify to WC > General Settings"</strong>

> <strong>[Upgrade to Pro!](https://plugin-devs.com/product/shopify-to-woocommerce-migration/)</strong>

==Features==
* <strong>Import Shopify Product: </strong> You can import all of your products from your Shopify Store to your WooCommerce store Easily as <strong>Simple</strong> or <strong>Variable</strong>. It will automatically detect the Product Type and import as <strong>Simple</strong> or <strong>Variable</strong> Product to your WooCommerce Store
* <strong>Import Shopify Product Images: </strong> You can Choose if you want to import your Product Images. <strong>Shopify to WooCommerce Migration </strong> plugin lets you import your products with product Images too. <strong>Shopify to WooCommerce Migration</strong> plugin will import all of the <strong>Gallery and Featured Images</strong> for your products.
* <strong>Import Shopify Categories: </strong> Shopify to WooCommerce Migration plugin will give you option to Import your <strong>Shopify Categories into WooCommerce Store</strong> and assign those categories automatically to their releavent products. You do not need to do assign products to imported categories after the Migration. <strong>Shopify to WooCommerce Migration</strong> plugin will do this automatically on behalf of you
* <strong>Import Shopify Categories Images: </strong> You can also Import your Shopify <strong>Category Images </strong> to WooCommerce Store
* <strong>Real time Import Logs: </strong> You can see the Migration Process on Real time. <strong>Shopify to WooCommerce Migration</strong> will Display the Progress of the Migratin on realtime so that you can see what is happening under the hood.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin folder to the '/wp-content/plugins/' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to "WordPress Admin > Shopify to WC"
